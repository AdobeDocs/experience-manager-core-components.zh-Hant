# 在 Adobe Experience Manager 自適應表單中啟用和使用文字輸入驗證模式

## 概觀

在 Adobe Experience Manager (Adobe Experience Manager) Forms 中，文字輸入元件的特定驗證格式 (如社會安全號碼、電子郵件地址或郵遞區號) 是在範本原則層級進行控制。本指南說明如何：

1. 存取範本編輯器，並設定文字輸入元件原則以公開驗證模式。
2. 建立自適應表單，並將這些驗證模式套用至文字輸入元件。

## 先決條件

- 存取已啟用 Forms 的 Adobe Experience Manager 執行個體。
- 編輯範本的權限 (通常是 `template-authors` 群組的一部分)。
- 建立和編輯自適應表單的權限。

## 第 1 部分：啟用範本中的驗證模式

>[!VIDEO](https://video.tv.adobe.com/v/3480390/)

### 步驟 1：存取範本主控台

1. 在 Adobe Experience Manager 首頁畫面中，按一下左側邊欄中的「**工具**」(槌子圖示)。
2. 導覽至「**一般 > 範本**」。
3. 選取包含專案範本的資料夾 (例如核心元件範例)。
4. 找到您要修改的特定範本 (例如 AF 空白) 並加以選取。
5. 按一下「**編輯**」(或直接開啟範本)。

### 步驟 2：存取元件結構

1. 確定您處於「**結構**」視圖 (如有必要，請從右上角的模式下拉式清單中選取「結構」)。
2. 找到允許文字輸入元件的主要&#x200B;**版面容器**&#x200B;或特定容器。
3. 按一下「容器」以顯示允許的元件清單。
4. 在清單中尋找「**自適應表單文字方塊**」元件。
5. 按一下元件名稱旁的「**原則**」圖示 (以檔案/抽屜符號表示)。

### 步驟 3：設定文字輸入元件原則

1. 在原則編輯器中，按一下視窗右側的「**格式**」索引標籤。
2. 您將看到可用驗證模式的清單。勾選您要啟用的格式方塊：
   - 電話號碼
   - 社會安全號碼
   - 電子郵件英數字元
   - 郵遞區號

### 步驟 4：定義並儲存原則

1. 切換回「**原則**」索引標籤 (或檢查左側的設定)。
2. 在「**原則標題**」欄位中，輸入清楚的名稱 (例如 `new-textinput-default-policy`)。
3. 按一下右上角的「**完成**」(核取記號) 圖示以儲存變更。

## 第 2 部分：建立表單並使用驗證模式

現在範本中已啟用驗證模式，您可以建立自適應表單並將其套用至文字輸入元件。

>[!VIDEO](https://video.tv.adobe.com/v/3480391/)

### 步驟 1：建立自適應表單

1. 導覽至「**表單與文件**」介面。
2. 按一下右上角的「**建立**」按鈕。
3. 從下拉選項中選取「**自適應表單**」。
4. 選擇您啟用驗證模式的範本 (例如&#x200B;**空白**)，然後按一下「**下一步**」。
5. 在「新增屬性」畫面上：
   - 輸入表單的&#x200B;**標題** (例如「testPolicy」)。「**名稱**」欄位會根據標題自動填入。
   - 請確定已選取正確的&#x200B;**主題用戶端程式庫** (例如 `adaptiveform.theme.canvas3`)。
6. 按一下「**建立**」。 此時會顯示成功訊息。
7. 按一下「**編輯**」，在編輯器中開啟表單。

### 步驟 2：新增文字輸入元件

1. 在表單編輯器中，在左側邊欄中找到「**元件**」瀏覽器。
2. 使用搜尋列尋找&#x200B;**自適應表單文字方塊**&#x200B;元件。輸入「文字」會篩選清單。
3. 按一下並將「**自適應表單文字方塊**」元件拖曳到主畫布區域。

### 步驟 3：設定元件格式

1. 按一下新增的&#x200B;**文字輸入**&#x200B;元件加以選取。
2. 按一下顯示在元件上方之工具列中的「**設定**」圖示 (以扳手表示)。
3. 在屬性對話框中，導覽至「**格式**」索引標籤。
4. 找到「**類型**」下拉式選單，並將選擇變更為「**電話號碼**」(或其他啟用的格式)。

### 步驟 4：設定資料驗證

1. 仍在設定對話框元件中時，導覽至「**驗證**」索引標籤。
2. 找到「**驗證模式**」區段。
3. 按一下「**類型**」下拉式選單。
4. 從驗證模式清單中選取「**電話號碼**」。這可確保在使用者輸入的文字不符合有效的電話號碼格式時，表單會產生錯誤。
5. 按一下對話框右上方的「**完成**」(核取記號) 圖示以儲存變更。

## 驗證

若要確認驗證模式是否正常運作：

1. 在編輯器中預覽表單。
2. 在「**文字輸入**」元件中輸入測試資料。
3. 確認：
   - 啟用的驗證模式會出現在元件的「格式」和「驗證」索引標籤中。
   - 無效的輸入會觸發適當的錯誤訊息。
   - 系統會接受有效的輸入，且不會出現錯誤。

## 疑難排解

- **問題：**&#x200B;驗證模式未出現在表單元件中。
   - **解決方案：**&#x200B;請確定範本原則已正確儲存，且您在建立表單時使用正確的範本。

- **問題：**&#x200B;無法存取範本編輯器。
   - **解決方案：**&#x200B;請驗證您擁有編輯範本的必要權限 (`template-authors` 群組的會員資格)。

- **問題：**&#x200B;文字輸入元件未正確驗證輸入。
   - **解決方案：**&#x200B;在元件的「驗證」索引標籤中重新檢查驗證模式設定，並確定選取了正確的模式。

## 後續步驟

在您的 Adobe Experience Manager 自適應表單中啟用並使用文字輸入驗證模式後：

- 為其他資料類型設定其他驗證模式。
- 探索其他元件原則以自訂表單行為。
- 設定表單提交處理和資料處理邏輯。
