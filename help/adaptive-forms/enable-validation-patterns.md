# 在Adobe Experience Manager Adaptive Forms中啟用和使用文字輸入驗證模式

## 概觀

在Adobe Experience Manager (Adobe Experience Manager) Forms中，文字輸入元件的特定驗證格式（如社保號碼、電子郵件地址或郵遞區號）會在範本原則層級進行控制。 本指南說明如何：

1. 存取範本編輯器，並設定文字輸入元件原則以公開驗證模式。
2. 建立最適化表單，並將這些驗證模式套用至文字輸入元件。

## 先決條件

- 存取已啟用Forms的Adobe Experience Manager執行個體。
- 編輯範本的許可權（通常是`template-authors`群組的一部分）。
- 建立和編輯Adaptive Forms的許可權。

## 第1部分：啟用範本中的驗證模式

>[!VIDEO](https://video.tv.adobe.com/v/3480390/)

### 步驟1：存取範本主控台

1. 在Adobe Experience Manager首頁畫面中，按一下左側邊欄中的&#x200B;**工具** （槌子圖示）。
2. 瀏覽至&#x200B;**一般>範本**。
3. 選取包含專案範本的資料夾（例如核心元件範例）。
4. 找到您要修改的特定範本（例如，「AF空白」）並選取它。
5. 按一下&#x200B;**編輯** （或直接開啟範本）。

### 步驟2：存取元件結構

1. 確定您處於&#x200B;**結構**&#x200B;檢視（如有必要，請從右上角的模式下拉式清單中選取「結構」）。
2. 找出允許文字輸入元件的主要&#x200B;**配置容器**&#x200B;或特定容器。
3. 按一下容器以顯示允許的元件清單。
4. 在清單中尋找&#x200B;**最適化表單文字方塊**&#x200B;元件。
5. 按一下元件名稱旁的&#x200B;**原則**&#x200B;圖示（以檔案/抽屜符號表示）。

### 步驟3：設定文字輸入元件原則

1. 在原則編輯器中，按一下視窗右側的&#x200B;**格式**&#x200B;索引標籤。
2. 您將看到可用驗證模式的清單。 勾選您要啟用的格式方塊：
   - 電話號碼
   - 社會安全號碼
   - 電子郵件英數字元
   - 郵遞區號

### 步驟4：定義並儲存原則

1. 切換回&#x200B;**原則**&#x200B;標籤（或檢查左邊的設定）。
2. 在&#x200B;**原則標題**&#x200B;欄位中，輸入清除的名稱（例如，`new-textinput-default-policy`）。
3. 按一下右上角的&#x200B;**完成** （核取記號）圖示以儲存您的變更。

## 第2部分：建立表單並使用驗證模式

現在範本中已啟用驗證模式，您可以建立最適化表單並將其套用至文字輸入元件。

>[!VIDEO](https://video.tv.adobe.com/v/3480391/)

### 步驟1：建立最適化表單

1. 導覽至&#x200B;**Forms和檔案**&#x200B;介面。
2. 按一下右上角的&#x200B;**建立**&#x200B;按鈕。
3. 從下拉式選項中選取&#x200B;**最適化表單**。
4. 選擇您啟用驗證模式的範本（例如，**空白**），然後按一下&#x200B;**下一步**。
5. 在新增屬性畫面上：
   - 為您的表單輸入&#x200B;**Title** （例如「testPolicy」）。 **Name**&#x200B;欄位會根據標題自動填入。
   - 請確定已選取正確的&#x200B;**主題使用者端資料庫** （例如`adaptiveform.theme.canvas3`）。
6. 按一下「**建立**」。 成功訊息隨即出現。
7. 按一下&#x200B;**編輯**&#x200B;在編輯器中開啟表單。

### 步驟2：新增文字輸入元件

1. 在表單編輯器中，在左側邊欄中找到&#x200B;**元件**&#x200B;瀏覽器。
2. 使用搜尋列尋找&#x200B;**最適化表單文字方塊**&#x200B;元件。 輸入「文字」會篩選清單。
3. 按一下並將&#x200B;**最適化表單文字方塊**&#x200B;元件拖曳到主畫布區域。

### 步驟3：設定元件格式

1. 按一下新新增的&#x200B;**文字輸入**&#x200B;元件以選取它。
2. 按一下元件上方出現的工具列中的&#x200B;**設定**&#x200B;圖示（以扳手表示）。
3. 在屬性對話方塊中，瀏覽至&#x200B;**格式**&#x200B;索引標籤。
4. 找到&#x200B;**型別**&#x200B;下拉式功能表，並將選取專案變更為&#x200B;**電話號碼** （或其他啟用的格式）。

### 步驟4：設定資料驗證

1. 當仍在元件設定對話方塊中，瀏覽至&#x200B;**驗證**&#x200B;標籤。
2. 找到&#x200B;**驗證模式**&#x200B;區段。
3. 按一下&#x200B;**型別**&#x200B;下拉式功能表。
4. 從驗證模式清單中選取&#x200B;**電話號碼**。 這可確保在使用者輸入的文字不符合有效的電話號碼格式時，表單會產生錯誤。
5. 按一下對話方塊右上角的&#x200B;**完成** （核取記號）圖示以儲存變更。

## 驗證

若要確認驗證模式是否正常運作：

1. 在編輯器中預覽表單。
2. 在&#x200B;**文字輸入**&#x200B;元件中輸入測試資料。
3. 確認：
   - 啟用的驗證模式會出現在元件的「格式」和「驗證」標籤中。
   - 無效的輸入會觸發適當的錯誤訊息。
   - 接受有效的輸入且沒有錯誤。

## 疑難排解

- **問題：**&#x200B;驗證模式未出現在表單元件中。
   - **解決方案：**&#x200B;請確定範本原則已正確儲存，而且您在建立表單時使用正確的範本。

- **問題：**&#x200B;無法存取範本編輯器。
   - **解決方案：**&#x200B;請確認您擁有編輯範本的必要許可權（`template-authors`群組的成員資格）。

- **問題：**&#x200B;文字輸入元件未正確驗證輸入。
   - **解決方案：**&#x200B;在元件的[驗證]索引標籤中重新檢查驗證模式設定，並確定選取的模式正確。

## 後續步驟

在您的Adobe Experience Manager Adaptive Forms中啟用並使用文字輸入驗證模式後：

- 為其他資料型別設定其他驗證模式。
- 探索其他元件原則以自訂表單行為。
- 設定表單提交處理和資料處理邏輯。
